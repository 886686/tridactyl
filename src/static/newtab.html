Tridactyl's Top Tips Page!

- Do not try to navigate to any about:* pages using :open. They are not useful for anything anyway.
- If you want your browser to look less ELC [1], the supplied userChrome-minimal.css to chrome/userChrome.css in your user profile folder, which can be find on the page about:support.
- Use yy to copy the current page URL to your clipboard (also works pages other than this).
- Press ]] and [[ to navigate comics and the like.
- Press s to search Google
- If it is raining outside, be sure to wear a raincoat.
- If you want your user interface to be less janky, consider using Firefox ESR [2] and Vimperator instead.

[1]: http://www.elc.co.uk/
[2]: https://www.mozilla.org/en-US/firefox/organizations/
